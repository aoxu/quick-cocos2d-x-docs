
## transition.moveBy()

> transition.moveBy(target, args)

Moves a sprite x,y points by modifying it's position attribute. x and y are relative to the position of the object.

### Example:

    -- Moving right and up
    transition.moveBy(sprite, {time = 2.0, x = 100, y = 100})

### Parameters:

-   CCNode **target**

-   table **args**
